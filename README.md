## Instructions to run

-   Pre-requisites:

    -   Node.js
    -   NPM
    -   Redis

-   Directions to setup/install

```bash
$ git clone https://github.com/dsckiet/website-backend-v2
$ cd website-backend-v2
$ touch .env
$ cp .env.example .env
# Set values in .env
$ npm i
```

-   Directions to execute

```bash
$ npm run dev
```
